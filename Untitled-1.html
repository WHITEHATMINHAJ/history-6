About This Project

Web Technologies Used

This website demonstrates the core web technologies that power the modern internet:

HTML

HTML (HyperText Markup Language) provides the structure and content of web pages. Originally developed by Tim Berners-Lee in the early 1990s, it has evolved through several versions [citation:2][citation:3].

CSS

CSS (Cascading Style Sheets) controls the visual presentation of web content. First proposed by Håkon Wium Lie in 1994, it allows separation of content from design [citation:6][citation:3].

JavaScript

JavaScript adds interactivity to web pages. Created by Brendan Eich in 1995, it has evolved from a simple scripting language to a powerful programming language used for both client-side and server-side development [citation:1][citation:5].

How They Work Together

These three technologies form the foundation of modern web development [citation:3][citation:8]:

  • HTML creates the structure (like the frame of a building)
  • CSS adds styling (like the paint and decorations)
  • JavaScript enables interactivity (like the electrical and plumbing systems)

This separation of concerns allows developers to create rich, interactive web experiences that work across different devices and platforms.
